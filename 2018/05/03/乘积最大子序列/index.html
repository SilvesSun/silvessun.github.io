题目

解析

前 n 位的乘积最大子序列分以下3种情况：

  1. 前n - 1位的乘积最小子序列的乘积 * A[n]最大

  2. 前n - 1位的乘积最大子序列的乘积 * A[n]最大

  3. 前n - 1位的乘积最大子序列和乘积最小子序列的乘积 * A[n]都不是最大，而A[n]本身最大

f_max(n)表示到原数组第n位时，乘积最大子序列的乘积，f_min(n)表示到原数组第n位时，乘积最小子序列的乘积
