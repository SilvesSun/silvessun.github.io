## 1 计算函数运行时间
```py
import time
from datetime import datetime as dt


def run_time(func):
    def wrapper(*args, **kwargs):
        start = dt.now()
        ret = func(*args, **kwargs)
        time.sleep(10)
        end = dt.now()
        print(end - start)
        return ret
    return wrapper
```

## 2 自定义输出内容
还是上面那个例子, 如果装饰器本身需要传入参数, 那就需要编写一个返回decorator的高阶函数，写出来会更复杂。

```py
import time
from datetime import datetime as dt


def log(text):
    def run_time(func):
        def wrapper(*args, **kwargs):
            start = dt.now()
            ret = func(*args, **kwargs)
            time.sleep(2)
            end = dt.now()
            print(text, (end - start).total_seconds())
            return ret
        return wrapper
    return run_time
```

## 3 装饰器实现单例
```py
def singleton(cls, *args, **kw):  
    instances = {}  
    def _singleton():  
        if cls not in instances:  
            instances[cls] = cls(*args, **kw)  
        return instances[cls]  
    return _singleton 
```
